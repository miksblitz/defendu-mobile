## Pose logic

Lead hook = **user’s left hand** (lead hand) throws the hook; **user’s right hand** stays in guard.

- In the app: “Your left” = MediaPipe **right** (camera mirror). So punch = MediaPipe right arm extending; guard = MediaPipe left arm contracted, wrist up.
- Same arm assignment as orthodox jab; hook is a curved punch so wrist–shoulder distance still increases at impact.
